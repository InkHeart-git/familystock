## 新增功能（参考贝莱德阿拉丁系统）

### 1. 风险仪表盘 🎯
- **风险分数仪表盘**: 0-100分实时显示
- **风险等级**: 极低/低/中/高/严重
- **风险热力图**: 8大行业风险热力分布
  - 科技、金融、能源、消费、医疗、军工、地产、材料

### 2. 投资组合风险敞口 📊
- **资产配置可视化**: 股票/债券/商品/现金
- **行业敞口分析**: 各行业配置比例和风险评分
- **悬停提示**: 显示详细信息

### 3. 压力测试 & P&L影响 ⚡
- **三情景分析**:
  - 🟢 乐观情景 (25%概率): +12.5%
  - 🔵 基准情景 (50%概率): -2.3%
  - 🔴 悲观情景 (25%概率): -18.7%
- **预期P&L**: 加权平均 -2.7%
- **风险指标**: VaR(95%): -8.5%, 最大回撤: -20%

### 4. AI智能选股 🤖
- **输入股票代码自动分析**
- **AI评分**: 0-100分综合评分
- **情景分析**: 乐观/基准/悲观情景P&L
- **风险指标**: VaR、最大回撤
- **AI投资建议**: 买入/增持/观望/减持

### 5. 实时预警系统 🚨
- **三级预警**:
  - 🔴 严重 (Critical)
  - 🟠 警告 (Warning)
  - 🟡 注意 (Caution)
- **预警面板**: 可折叠的顶部预警面板
- **推送通知**: 实时通知新预警
- **预警详情**: 点击查看详细推演结果

## API端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/` | GET | 系统信息 |
| `/api/v3/pipeline/status` | GET | 流水线状态 |
| `/api/v3/pipeline/run` | POST | 手动触发流水线 |
| `/api/v3/alerts` | GET | 预警列表 |
| `/api/v3/alerts/active` | GET | 活跃预警 |
| `/api/v3/dashboard` | GET | 风险仪表盘 |
| `/api/v3/news` | GET | 新闻列表 |
| `/api/v3/stock/analyze` | POST | 股票分析 |

## 技术特点

1. **全自动运行**: 无需手动输入，系统自动采集分析
2. **实时更新**: 自动刷新数据，无需手动刷新页面
3. **参考阿拉丁**: 风险指标、压力测试、情景分析
4. **模块化设计**: 新增功能以面板形式集成，不影响原有布局
5. **响应式设计**: 适配移动端和桌面端

## 使用说明

### 查看风险仪表盘
1. 打开 http://43.160.193.165/v3/
2. 首页显示风险仪表盘、热力图、资产敞口
3. 查看压力测试和P&L影响分析

### 分析股票
1. 点击底部"选股"标签
2. 输入股票代码（如：600519）
3. 点击搜索，查看AI推演分析结果

### 查看预警
1. 点击顶部预警面板展开
2. 或点击底部"预警"标签查看全部预警
3. 点击预警查看详细推演结果

## 注意事项

1. API服务器运行在8080端口
2. 首次加载可能需要等待AI推演完成
3. 预警数据每5分钟自动更新
4. 建议定期运行测试脚本验证API状态
