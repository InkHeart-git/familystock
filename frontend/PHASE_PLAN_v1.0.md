## 🎯 项目概述

MiniRock 是一款面向个人投资者的智能投资终端，提供A股实时行情、AI分析、持仓管理和预警功能。

**测试环境**: http://43.160.193.165/minirock-v2.html

---

## 📋 Phase 1: 基础功能 (已完成)

### 交付目标
建立核心持仓管理和实时行情系统

### 功能清单

| 功能 | 状态 | 验收标准 |
|------|------|---------|
| 持仓列表显示 | ✅ | 显示股票名称、代码、数量、价格、涨跌幅、盈亏 |
| 添加持仓 | ✅ | 支持输入持股数量和持仓成本 |
| 删除持仓 | ✅ | 点击删除按钮，确认后删除 |
| 实时行情 | ✅ | 接入Tushare API，显示真实股价 |
| 个股详情页 | ✅ | 显示股票基本信息和AI建议 |
| 股票搜索 | ✅ | 支持名称/代码搜索，7368只股票数据库 |

### API端点

```
GET  /api/portfolio/holdings      # 获取持仓列表
POST /api/portfolio/holdings      # 添加持仓
DELETE /api/portfolio/holdings/{symbol}  # 删除持仓
GET  /api/portfolio/alerts        # 获取预警信息
GET  /api/tushare/quote/{code}    # 获取实时行情
GET  /api/tushare/search?q={query} # 搜索股票
```

### 数据持久化
- ✅ PostgreSQL数据库
- ✅ 持仓数据持久化存储
- ✅ 重启后数据不丢失

### 预警系统
- ✅ 基于持仓盈亏自动生成预警
- ✅ 亏损超过10%：高风险预警
- ✅ 亏损超过5%：中风险预警
- ✅ 盈利超过20%：止盈提醒
- ✅ 今日大涨/大跌提醒

### 测试文档
- [Phase 1 测试用例](http://43.160.193.165/TEST_PHASE1_v1.0.md)

## 📋 Phase 2: AI分析增强 (待开发)

### 交付目标
接入真实AI分析，提供个性化投资建议

### 功能清单

| 功能 | 状态 | 验收标准 | 预计工时 |
|------|------|---------|---------|
| AI个股诊断 | ⏳ | 调用AI服务生成个性化报告 | 4小时 |
| 组合分析 | ⏳ | 持仓健康度、风险评估、配置建议 | 3小时 |
| 新闻关联 | ⏳ | 显示与持仓股票相关的新闻 | 3小时 |
| 事件影响分析 | ⏳ | 重大事件对个股的影响评估 | 4小时 |
| 情景推演 | ⏳ | 模拟不同市场情况下的盈亏 | 4小时 |

### API需求

```
POST /api/ai/analyze-stock        # AI分析个股
POST /api/ai/analyze-portfolio    # 组合分析
GET  /api/news/related/{symbol}   # 相关新闻
POST /api/ai/scenario             # 情景推演
```

### 技术依赖
- AI分析服务 (Kimi API)
- 新闻采集系统 (Mediastack)
- 事件数据库
