## 🔴 P1 - Magic Memorial (魔法纪念)

### 项目概述
AI驱动的电子祭拜平台，让逝者照片"活"起来 (类似哈利波特预言家日报效果)

### 核心价值主张
- **情感价值**: 让亲人"重现"，缓解思念
- **技术价值**: AI + 视频生成，创新祭拜体验
- **文化价值**: 传统+科技，传承记忆

### 目标市场
- 国内市场: 清明节、中元节、重阳节
- 海外市场: 新加坡、美国华人社区
- 合作渠道: 殡仪馆、墓园、宗教场所

### 商业模式
| 服务 | 价格 | 说明 |
|------|------|------|
| 免费试用 | ¥0 | 5次生成额度 |
| 月度会员 | ¥19.9/月 | 无限生成 + 云存储 |
| 年度会员 | ¥199/年 | 优惠20% + 专属客服 |
| 虚拟祭品 | ¥5-50/个 | 鲜花、香烛、贡品等 |
| 定制服务 | ¥299起 | 高端定制视频 |

### 技术方案
- **视频生成**: 阿里云 LivePortrait + EMO
- **成本**: ¥0.1-0.4/视频
- **服务器**: 新加坡 (TenClaw)
- **存储**: 阿里云OSS + CDN加速

## 🟡 P2 - AI小说创作工具

### 项目概述
基于《十日终焉》创作经验，打造专业网文创作辅助工具

### 核心功能
1. **世界观生成器**
   - 输入关键词自动生成完整世界观
   - 地图、势力、历史背景

2. **人物关系图谱**
   - 可视化人物关系
   - 自动生成人物小传
   - 性格冲突分析

3. **剧情推演引擎**
   - 输入大纲自动扩展章节
   - 多线剧情管理
   - 伏笔追踪

4. **智能润色**
   - 现代词汇替换古风词
   - 对话优化
   - 节奏把控

5. **多平台发布**
   - 一键生成起点/番茄/晋江格式
   - 自动排版
   - 字数统计

### 目标用户
- 新手网文作者 (核心)
- 职业编剧
- 自媒体内容创作者

### 商业模式
| 版本 | 价格 | 功能 |
|------|------|------|
| 免费版 | ¥0 | 基础生成，每日限额 |
| 专业版 | ¥49/月 | 无限生成，高级功能 |
| 团队版 | ¥199/月 | 多账号，协作功能 |
| 定制版 | 议价 | 企业定制API |

### 技术方案
- **AI模型**: Kimi/GPT-4
- **前端**: Vue3 + TypeScript
- **后端**: FastAPI
- **数据库**: PostgreSQL
