## 🎨 设计规范

### 颜色系统

```css
:root {
  /* 背景 */
  --bg-primary: #0a0a0f;        /* 主背景 - 深邃黑 */
  --bg-secondary: #1a1a2e;      /* 次级背景 */
  --bg-card: #16213e;           /* 卡片背景 */
  
  /* 文字 */
  --text-primary: #ffffff;      /* 主文字 */
  --text-secondary: #8892b0;    /* 次级文字 */
  --text-muted: #64748b;        /* 辅助文字 */
  
  /* 强调色 */
  --accent-up: #ff6b6b;         /* 上涨 - 红 */
  --accent-down: #51cf66;       /* 下跌 - 绿 */
  --accent-blue: #3b82f6;       /* 主蓝 */
  --accent-purple: #8b5cf6;     /* 紫色 */
  --accent-yellow: #f59e0b;     /* 黄色 */
  
  /* 渐变 */
  --gradient-primary: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  --gradient-success: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);
  --gradient-danger: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
}
```

### 字体规范

| 用途 | 大小 | 字重 | 行高 |
|------|------|------|------|
| 页面标题 | 20px | 700 | 1.2 |
| 卡片标题 | 16px | 600 | 1.3 |
| 正文字号 | 14px | 400 | 1.5 |
| 辅助文字 | 12px | 400 | 1.4 |
| 数字金额 | 28-42px | 700 | 1.1 |

### 间距系统

```
xs:  4px
sm:  8px
md:  16px
lg:  24px
xl:  32px
2xl: 48px
```

### 圆角规范

```
小标签: 4px
按钮:   8px
卡片:   16px
大卡片: 24px
圆形:   50%
```

## 📲 交互规范

### 手势操作

| 手势 | 操作 | 触发区域 |
|------|------|----------|
| 下拉 | 刷新数据 | 列表顶部 |
| 上拉 | 加载更多 | 列表底部 |
| 左滑 | 显示操作 | 持仓卡片 |
| 长按 | 批量选择 | 列表项 |
| 双击 | 快速添加 | 行情列表 |

### 动画时长

```
微交互: 150ms (按钮点击)
过渡动画: 300ms (页面切换)
加载动画: 800ms+ (AI分析)
```
