# FamilyStock Prototype

股票分析与全球投资情报中心网页原型

## 文件说明

| 文件 | 说明 | 大小 |
|------|------|------|
| `ai-stock-picker.html` | AI智能选股系统 | 39KB |
| `global-intelligence-hub-v2.html` | 全球投资中心V2（融合版） | 57KB |
| `global-intelligence-hub.html` | 全球情报中心原版 | 53KB |

## 功能特性

### AI智能选股系统
- A股实时数据展示
- AI多因子选股评分
- 策略配置中心
- 股票详情弹窗

### 全球投资中心V2
- 3D地球可视化（NASA真实纹理）
- 全球冲突热点标记
- 大宗商品期货
- AI选股 + 财经日历
- 大V观点实时追踪

## 技术栈
- HTML5 + Tailwind CSS (CDN)
- Three.js (3D地球)
- Chart.js (图表)
