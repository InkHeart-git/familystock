## Bug 列表

### Bug #1: 组合分析页面加载失败
- **严重程度**: 🔴 高
- **问题描述**: 访问 portfolio-analysis.html 显示"加载失败"，点击重试按钮无效
- **复现步骤**: 
  1. 访问 https://minirock.online/portfolio-analysis.html
  2. 页面显示"加载失败"
  3. 点击"重试"按钮，问题依旧
- **建议**: 检查后端 API 接口是否正常

### Bug #2: 自选股/市场行情页面显示异常
- **严重程度**: 🟡 中
- **问题描述**: 访问 watchlist.html 和 market.html 都显示首页内容而非对应功能页面
- **复现步骤**: 
  1. 访问 https://minirock.online/watchlist.html
  2. 页面显示首页内容而非自选股列表
- **建议**: 检查路由配置或页面文件

### Bug #3: 我的持仓数据加载中
- **严重程度**: 🟡 中
- **问题描述**: portfolio.html 页面持续显示"正在加载持仓数据..."
- **建议**: 检查持仓数据 API 接口

### Bug #4: 搜索功能可能受影响
- **严重程度**: 🟢 低
- **问题描述**: 输入股票代码后按回车未跳转，可能受风险提示弹窗影响
- **建议**: 检查搜索功能和风险提示的交互逻辑
