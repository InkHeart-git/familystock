## 🚀 快速开始

### 安装依赖（仅需一次）

```bash
cd /var/www/familystock/tests/browser-automation
npm init -y
npm install playwright
```

### 运行测试

```bash
# 一键完整测试
./run-tests.sh full

# 仅测试认证
./run-tests.sh auth

# 仅测试持仓
./run-tests.sh portfolio

# 生成 HTML 报告
./run-tests.sh report
```

## 🔧 技术亮点

### 反检测特性
- ✅ 贝塞尔曲线鼠标轨迹
- ✅ 逐字输入（带随机延迟）
- ✅ 3% 概率模拟 typo 并修正
- ✅ 随机页面浏览（滚动+鼠标移动）

### 零 Token 成本
- 直接连接浏览器 CDP
- 不调用 OpenClaw API
- 可无限次重复运行

### 报告生成
- JSON 结构化报告

- 历史记录保存
