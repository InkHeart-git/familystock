# 魔法纪念 - 产品需求文档 (PRD)

**文档版本**: v1.0  
**创建日期**: 2026-03-07  
**作者**: 玲珑  
**产品名称**: 魔法纪念 (Magic Memorial)  
**产品定位**: AI驱动的电子祭拜平台，让逝去的亲人以"魔法照片"的形式永远陪伴

---

## 1. 产品概述

### 1.1 产品背景
传统在线祭祀平台只提供静态纪念馆和简单的虚拟祭品。随着AI视频生成技术的成熟，我们可以让逝者的照片"活"起来，像《哈利波特》中的预言家日报一样，在画面中微笑、走动、互动，给用户更深层的情感慰藉。

### 1.2 产品愿景
让每一次祭拜都成为一次温暖的"重逢"，用科技延续记忆的温度。

### 1.3 目标用户
- 无法亲自前往墓地祭拜的海外华人
- 工作繁忙、时间有限的都市白领
- 希望为逝去宠物/亲人建立数字纪念的年轻群体
- 重视环保、接受新型祭祀方式的现代人

## 2. 核心功能

### 2.1 主题选择系统 (入口功能)

#### 功能描述
用户创建纪念馆时，首先选择视觉主题，系统将基于主题生成全套魔法场景和交互体验：

| 主题 | 东方天宫 | 西方魔法 |
|------|---------|---------|
| **视觉风格** | 云雾缭绕的仙境、金色祥云、琼楼玉宇 | 霍格沃茨城堡、魔法书架、飘浮蜡烛 |
| **色调** | 金、红、白、青 | 暖黄、深棕、墨绿、古铜 |
| **BGM** | 古筝、笛子、编钟 | 管弦乐、竖琴、神秘吟唱 |
| **照片边框** | 祥云纹、龙凤图腾 | 复古相框、魔法符文 |
| **粒子特效** | 花瓣飘落、仙气缭绕 | 萤火虫、魔法粉尘、飘雪 |

#### 主题影响范围
- 魔法照片的背景场景
- 虚拟祭坛的3D环境
- 祭品的视觉样式（中式香烛 vs 西式蜡烛）
- 转场动画和特效
- 系统提示语的文风（古风 vs 奇幻）

### 2.2 魔法照片生成 (核心功能)

#### 功能描述
用户上传1-3张逝者照片，AI自动生成多个场景的"魔法视频"，场景内容根据所选主题变化：

**东方天宫主题场景**：

| 场景类型 | 动作描述 | 背景元素 | 情感定位 |
|---------|---------|---------|---------|
| 微笑场景 | 人物微笑、眨眼、微微点头 | 云海、仙鹤、天宫远景 | 仙界安宁 |
| 挥手场景 | 人物挥手告别/致意 | 彩虹桥、祥云流动 | 天各一方 |
| 生活场景 | 人物品茶、看书、赏花 | 亭台楼阁、蟠桃树 | 仙居日常 |
| 离开场景 | 人物乘云而去，渐隐天际 | 日出东方、金光大道 | 羽化登仙 |
| 节日场景 | 春节作揖、中秋赏月 | 灯笼、满月、烟花 | 天人共庆 |

**西方魔法主题场景**：

| 场景类型 | 动作描述 | 背景元素 | 情感定位 |
|---------|---------|---------|---------|
| 微笑场景 | 人物微笑、眨眼、微微点头 | 霍格沃茨大礼堂、飘浮蜡烛 | 魔法温情 |
| 挥手场景 | 人物挥手告别/致意 | 9¾站台、霍格沃茨特快 | 魔法启程 |
| 生活场景 | 人物读书、魔药调配、逗猫 | 魔法书架、坩埚、猫头鹰 | 巫师日常 |
| 离开场景 | 人物走进画像框、化为光点 | 邓布利多办公室、冥想盆 | 魔法永存 |
| 节日场景 | 圣诞节举杯、万圣节南瓜 | 圣诞树、南瓜灯、魔法雪 | 魔法庆典 |

#### 技术要求
- 基于 OmniHuman-1 / EMO / Runway Gen-4 等AI视频生成模型
- 单场景视频时长：5-10秒循环
- 输出分辨率：720p（免费版）/ 1080p（付费版）
- 支持格式：MP4（后台自动转码为HLS）

### 2.2 在线纪念馆

#### 功能描述
每个用户可创建多个纪念馆：
- 纪念馆信息：姓名、生卒年月、生平简介
- 照片墙：上传多张纪念照片
- 留言板：亲友可留言悼念
- 访问权限：公开/私密/密码保护

#### 功能描述
提供丰富的虚拟祭品，根据主题呈现不同的视觉风格和仪式体验：

#### 仪式流程
1. 选择主题（首次创建时确定，可后续切换）
2. 选择祭品（主题化视觉呈现）
3. 播放祭拜动画（魔法照片在主题背景中）
4. 用户可留言/祈福（主题化文字模板）
5. 生成主题风格分享卡片

- 祭品拖拽摆放

- 生成分享卡片

## 4. 商业模式

### 4.1 收入结构

```
总收入
├── 订阅收入 (60%)
│   ├── 月度会员 ¥19.9/月
│   └── 年度会员 ¥168/年
├── 虚拟祭品 (35%)
│   ├── 基础祭品 ¥3-9.9
│   ├── 进阶祭品 ¥19.9-49.9
│   └── 豪华套餐 ¥99-199
└── 增值服务 (5%)
    ├── 定制视频 ¥299起
    └── 二维码铜牌 ¥99
```

### 4.2 成本结构

| 成本项 | 单价 | 占比 |
|--------|------|------|
| AI视频生成 | ¥2-5/次 | 40% |
| CDN带宽 | ¥0.1/GB | 20% |
| 云存储 | ¥0.12/GB/月 | 10% |
| 支付手续费 | 0.6% | 5% |
| 运营推广 | - | 25% |

### 4.3 盈利模型

- **单用户获取成本 (CAC)**: ¥15-30
- **用户生命周期价值 (LTV)**: ¥150-300
- **LTV/CAC 比率**: 5-10x（健康水平）

## 5. 技术需求

### 5.1 性能要求

| 指标 | 目标值 |
|------|--------|
| 视频生成时间 | < 2分钟/场景 |
| 视频首帧加载 | < 1秒 |
| 页面加载时间 | < 2秒 |
| 并发视频播放 | 支持1000+ |
| 系统可用性 | 99.9% |

### 5.2 安全需求

- 视频内容加密（HLS AES-128）
- 播放权限验证
- 用户数据隐私保护
- 防刷机制（播放次数限制）

### 5.3 合规需求

- 殡葬管理条例合规
- 个人信息保护法
- 内容审核机制
- 用户举报功能
