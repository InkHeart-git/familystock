## 一、背景与定位

### 1.1 市场需求
中老年散户（40-70岁）是A股市场最大的散户群体，但几乎没有任何产品专门服务他们：
- **竞品空白**：雪球/同花顺面向年轻和专业用户，无针对中老年设计的产品
- **真实痛点**：信息过载（35%）、情绪化交易（25%）、诈骗风险（15%）、缺乏陪伴（20%）
- **核心洞察**：「情绪价值 > 分析价值」——他们最需要的不是"分析"，是"有人告诉我该怎么办"

### 1.2 产品定位
> **「AI投资陪伴」——不说废话，直接告诉你该怎么做**

定位：MiniRock 的第二个核心产品（与「AI股神争霸赛」并列）
目标用户：中老年散户（40-70岁），有一定炒股经验但缺乏专业分析能力

### 1.3 与现有系统的关系
```
MiniRock 主站
├── AI股神争霸赛（围观AI竞技，竞猜押注）  ← 现有
└── AI投资陪伴（个人化投资陪伴）           ← 新增
```

## 二、核心功能设计

### 2.1 三选一直接建议（核心）
用户用自然语言提问，AI 直接给结论：

**交互格式：**
```
用户：「我持有平安银行亏了20%，怎么办？」

AI 投资陪伴：
┌─────────────────────────────┐
│ 🎯 建议操作：持有             │
│ 📊 平安银行 | 现价12.30元     │
│ 📈 回本需涨：+25.0%          │
│ 🎯 目标价：13.50元           │
│ 🔻 止损位：11.20元           │
│ 🤖 置信度：82%              │
└─────────────────────────────┘
```

**不同于现有AI聊天：**
| | 现有AI聊天 | AI投资陪伴 |
|---|---|---|
| 回答风格 | 分析报告，几百字 | 三句话，结论先行 |
| 互动方式 | 泛泛而谈 | 情绪安抚优先 |
| 工具集成 | 无 | 回本计算器+语音 |
| 目标用户 | 专业投资者 | 中老年散户 |

### 2.2 语音播报（老年人刚需）
- 一键播报分析结论，语速可调（0.8x / 1.0x / 1.2x）
- 适配场景：老人看不清屏幕时，让子女帮忙操作
- 技术：Web Speech API（无需额外API）

### 2.3 回本计算器
```
亏损计算器：
- 输入：持仓成本、当前价格
- 输出：
  「亏损-20%，回本需涨+25%」
  「方案A：持有等反弹，约需X天」
  「方案B：补仓X手，成本降至Y元」
```

### 2.4 情绪陪伴模式
```
用户：「亏了很多，睡不好觉」

AI：「我理解你的心情，先深呼吸...
     亏损时最重要的是稳住，别恐慌割肉。
     你亏了多少？持仓是什么？」
```
**规则**：检测到焦虑/恐慌关键词，先安抚情绪（1-2句），再给分析。

### 2.5 持仓诊断
用户输入持仓股数和成本，AI 输出：
1. 当前亏损/盈利金额
2. 回本路径（几个方案）
3. 与大盘对比（跑赢/跑输指数）
4. 操作建议（加仓/持有/减仓/止损）

## 三、用户画像

### Persona 1：老王（60岁，退休散户）⭐⭐⭐⭐⭐
- **特征**：退休金炒股，亏损焦虑，视力下降，操作不熟练
- **核心场景**：亏钱了很慌，需要有人说"别怕，有办法"
- **使用频率**：每天看持仓，睡不好时问AI
- **支付意愿**：低，愿为免费功能留下

### Persona 2：李阿姨（50岁，半职投资者）⭐⭐⭐⭐
- **特征**：工作忙，快速获取操作建议
- **核心场景**：每天早上花2分钟看当日建议
- **使用频率**：交易日每天
- **支付意愿**：中，VIP会员可接受

### Persona 3：张先生（45岁，高端客户）⭐⭐
- **特征**：资产配置需求，个性化服务
- **核心场景**：组合诊断 + AI分身
- **使用频率**：定期，非每日
- **支付意愿**：高，可接受付费订阅

## 五、商业模式

### 5.1 免费版（引流）
- 每天3次AI陪伴提问
- 基础持仓诊断
- 语音播报

### 5.2 VIP会员（付费）
- 无限次提问
- 高级回本计算
- 个股深度分析报告
- 定价参考：¥9.9/月 或 ¥99/年

### 5.3 积分系统（复用）
与AI股神争霸共用积分系统：
- 每日签到+5积分
- 发帖+2积分
- 竞猜消耗积分
- 积分可兑换VIP天数

## 六、合规说明

1. **所有AI输出必须标注**：「AI分析仅供参考，不构成投资建议」
2. **高风险操作提示**：止损、割肉等决策前增加确认步骤
3. **禁止行为**：
   - 不预测具体价格（只能说区间）
   - 不保证收益
   - 不推荐涨停股（高风险标注）
4. **防诈骗提示**：识别到"内幕消息""老师带单"等关键词时，主动警示

## 九、与现有Phase 4/5的关系

AI投资陪伴是Phase 4/5的用户入口：
- **Phase 4（榜一大哥）**：AI投资陪伴用户 → 参与竞猜 → 成为榜一大哥 → 影响AI操盘
- **Phase 5（AI分身）**：榜一大哥 → 创建AI分身 → 自定义投资风格

也就是说：
> 新用户通过「AI投资陪伴」进入 → 被AI吸引 → 参与「AI股神争霸赛」竞猜 → 成为榜一大哥 → 最终创建自己的AI分身

**这是整个产品体系的自然增长飞轮。**
