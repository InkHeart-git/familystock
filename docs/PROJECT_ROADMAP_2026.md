# minirock.online 完整项目路线图
**整理时间**: 2026-04-22
**整理人**: PM

---

## 一、项目定位

**minirock.online** = MiniRock 主站 + AI股神争霸赛（子模块）

- **MiniRock 主站**: AI驱动的个人股票分析助手，定位"帮助老年人理解股票"
- **AI股神争霸赛**: minirock.online 下的 AI 交易竞技板块，10个 AI 实时实盘比赛
- **魔法纪念**（未来）: AI驱动的电子祭拜平台，独立产品

## 二、当前架构

### 服务端口

| 服务 | 端口 | 说明 |
|------|------|------|
| Nginx | 80/443 | 主站入口 |
| MiniRock API | 8000 | 生产 API |
| AI股神争霸 | 8084→18085 | API服务 |
| MiniRock 测试 | 8080/8001 | 测试环境 |

### 项目目录

```
/var/www/familystock/           # MiniRock 主站
├── frontend/                   # 前端HTML (index.html/minirock-v2.html等)
├── api/                        # FastAPI后端 (端口8000)
│   ├── app/routers/           # 路由 (ymos.py/chat.py等)
│   ├── data/family_stock.db   # SQLite数据库
│   └── tiered_sync_service.py # 分层实时同步 (5秒/30秒)
└── data/                       # 共享数据

/var/www/ai-god-of-stocks/      # AI股神争霸
├── api_server.py               # 手写HTTP服务器 (端口18085)
├── engine/                     # 10个AI大脑
│   ├── unified_scheduler.py    # 统一调度器
│   └── brains/                 # 具体AI实现
├── data/ai_god.db             # SQLite数据库
├── frontend/ai-god-competition.html  # 赛季页面
└── shared/                     # 共享记忆系统

/var/www/pm-console/            # PM任务管理
/root/.hermes/                 # Hermes Agent
```

### 数据库

| 数据库 | 位置 | 用途 |
|--------|------|------|
| family_stock.db | /var/www/familystock/api/data/ | MiniRock主站 |
| ai_god.db | /var/www/ai-god-of-stocks/data/ | AI股神争霸 |
| news.db | /var/www/familystock/api/data/ | 新闻数据 |
| memory.db | /var/www/ai-god-of-stocks/shared/ | 共享记忆 |

### 核心技术栈

- **前端**: HTML5 + Tailwind CSS + Vanilla JS
- **后端**: Python 3.12 + FastAPI + 手写BaseHTTPRequestHandler
- **AI模型**: MiniMax-M2.7-highspeed (主) → Kimi (备) → DeepSeek (终)
- **数据库**: SQLite + PostgreSQL (备用)
- **实时同步**: 东方财富API (无限制) + Tushare Pro (历史)
- **服务器**: 腾讯云新加坡 43.160.193.165
