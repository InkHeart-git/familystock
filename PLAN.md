# 家庭选股软件企划案

## 一、项目概述

### 1.1 项目背景
为家人定制开发一款智能化选股辅助工具，结合AI分析能力和实时数据搜索，帮助家庭成员做出更明智的投资决策。

### 1.2 核心定位
- **使用对象**：家庭成员（不超过10人）
- **部署方式**：私有部署，不对外发布
- **技术架构**：Web应用（优先）+ 可选安卓APK
- **核心能力**：AI智能分析 + 实时数据搜索

### 1.3 核心目标
1. 自动化股票筛选和初步分析
2. AI辅助解读财报、新闻和市场情绪
3. 风险提示和投资建议
4. 家庭共享自选股池

## 三、技术架构

### 3.1 推荐方案：Web应用（优先）

**理由：**
- 跨平台（手机、平板、电脑都能用）
- 部署简单（可部署在家里NAS或云服务器）
- 维护成本低
- 可后续打包成PWA或APP

### 3.2 技术栈

#### 前端
| 技术 | 用途 | 备选 |
|------|------|------|
| Vue 3 + TypeScript | 主框架 | React |
| Element Plus | UI组件库 | Ant Design |
| ECharts | 图表（K线、指标）| TradingView |
| Pinia | 状态管理 | Vuex |

#### 后端
| 技术 | 用途 | 备选 |
|------|------|------|
| Node.js + Express | API服务 | Python FastAPI |
| PostgreSQL | 主数据库 | MySQL |
| Redis | 缓存 | - |
| Python + FastAPI | AI服务 | Node.js |

#### AI能力
| 来源 | 用途 | 成本 |
|------|------|------|
| Kimi API | 财报解读、研报摘要 | 按量付费 |
| 本地Ollama | 简单分析（离线可用）| 免费 |
| OpenAI API | 复杂推理（备选）| 按量付费 |

### 3.3 数据接口

| 数据类型 | 推荐来源 | 费用 |
|---------|---------|------|
| 实时行情 | AkShare / Tushare | 免费-付费 |
| 历史K线 | Baostock / YFinance | 免费 |
| 财务数据 | AkShare | 免费 |
| 新闻资讯 | 新浪财经API | 免费 |
| 研报数据 | 东方财富Choice | 付费 |

## 五、AI能力设计

### 5.1 AI分析场景

| 场景 | 输入 | AI输出 | 调用频率 |
|------|------|--------|---------|
| 财报解读 | PDF财报/财报文本 | 3-5个核心要点 | 季度 |
| 研报摘要 | 研报PDF/链接 | 投资逻辑、目标价、风险提示 | 实时 |
| 新闻情绪 | 新闻标题+摘要 | 情绪评分(-1~+1)、关键词 | 实时 |
| 个股问答 | 用户自然语言问题 | 结构化回答 | 按需 |
| 选股策略 | 筛选条件+市场数据 | 符合条件的股票列表+推荐理由 | 按需 |

### 5.2 AI提示词设计（示例）

**财报解读Prompt：**
```
你是一位专业的财务分析师。请分析以下财报数据，输出：
1. 营收和利润增长情况（同比/环比）
2. 核心业务亮点
3. 潜在风险点
4. 现金流健康状况
5. 估值水平评价

财报数据：
{financial_data}

请用简洁的语言输出，适合非专业人士理解。
```

## 六、数据安全与隐私

### 6.1 安全措施
- ✅ 仅内网访问（家庭WiFi）或VPN访问
- ✅ 不需要用户注册（预设家庭成员账号）
- ✅ 无敏感个人信息收集
- ✅ 数据存储在本地/私有服务器

### 6.2 合规性
- ⚠️ 不对外提供服务，仅供家庭成员使用
- ⚠️ 数据来自公开渠道，不涉及内幕信息
- ⚠️ AI分析仅供参考，不构成投资建议
