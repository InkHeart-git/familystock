### BE-003: 密码重置 API（短信验证码方案）
**需求背景**:  
用户忘记密码时无法找回，需实现重置功能

**API 设计**:

#### 1. 发送短信验证码
```http
POST /api/auth/send-sms-code
Content-Type: application/json

{
  "phone": "13800138111"
}
```

**响应**:
```json
{
  "success": true,
  "message": "验证码已发送"
}
```

#### 2. 验证验证码并重置密码
```http
POST /api/auth/reset-password
Content-Type: application/json

{
  "phone": "13800138111",
  "code": "123456",
  "new_password": "NewPass123"
}
```

**实现要求**:
- 验证码 6 位数字，有效期 5 分钟
- 同一手机号每分钟限发 1 次，每天限 10 次
- 接入阿里云/腾讯云短信服务
