**问题描述**:

**复现步骤**:
1. 访问 /minirock-auth.html
2. 输入手机号: 13800138111
3. 输入密码: Test123456
4. 点击"立即登录"

**预期结果**: 正常提交表单，进入系统  
**实际结果**: 表单验证失败，无法提交

- 表单字段 name 属性缺失
- JavaScript 验证逻辑错误

**修复建议**:
```javascript

const form = document.getElementById('loginForm');
form.addEventListener('submit', (e) => {
  e.preventDefault();
  const phone = document.getElementById('phone').value;
  const password = document.getElementById('password').value;
  // 确保能正确获取值
  console.log(phone, password); // 调试
});
```
